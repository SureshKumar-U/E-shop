# E-shop

E-shop is an e-commerce application built with a Node.js backend and a React frontend.

## Features

- User authentication (login and signup)
- Product listing
- Product details
- Shopping cart

## Technologies Used
- Frontend: React, RTK Query, TailwindCSS,
- Backend: Node.js, Express.js
- Database: MongoDB
- Payment: Stripe API
